Privacy Policy

BusyShark Agency (“we”, “us”) respects your privacy.

Data we process

Our app uses the WhatsApp Cloud API to send and receive WhatsApp messages initiated by users. We process phone numbers, message content, and message metadata only to provide the messaging functionality.

Data sharing

Data is shared only with Meta Platforms, Inc. via the WhatsApp Cloud API and is not sold to third parties.

Data retention

Message data is retained only as long as necessary for the operation of the service or as required by law.
